# 抖音脚本 · Nomi 桌面版上线

> 风格：短、快、画面感强；前 3 秒必须钩住；口播自然，不要"打字稿"感。

| 3-7s | 录屏：在 ChatGPT 复制脚本 | "脚本写在这里" |
| 7-11s | 切到即梦，粘贴 prompt 等 loading | "图生在这里" |
| 11-15s | 切到 Runway 上传图等待 | "视频生在这里" |
| 15-19s | 切到剪映拖素材 | "剪辑还得换地方" |
| 19-22s | 全屏一个红色"X"叠加 | "搬来搬去，一天没了" |
| 22-26s | 切到 Nomi 工作台，一段剧本，一键拆镜头 | "所以我自己做了一个" |
| 26-30s | 画布上 6 个节点并行生成 | "开源、本地跑、双击就能用。简介区取" |

**底部字幕固定**：搜「Nomi」GitHub · 完全免费

## 脚本 2 · 实操演示款（60 秒，全程录屏 + 字幕，无人声更好做）

**目标**：让人看见"真的能跑通"。
**封面**：录屏中段截图，画布上一堆节点 + 大字"AI 帮我做完了"

**节奏：** 屏幕录制全程 1.5 倍速 + 节拍音乐 + 字幕。

| 时间 | 画面 | 字幕 |
|------|------|------|
| 0-3s | 打开 Nomi 桌面应用 | 「实测：用 AI 做一条广告片需要多久」 |
| 3-8s | 在创作区写下产品 + 7 段镜头 | 「Step 1 写脚本（手动，2 分钟）」 |
| 8-12s | 点「让 Agent 拆镜头」，画布上自动出 7 个节点 | 「Step 2 AI 拆镜头（自动，10 秒）」 |
| 12-25s | 7 个图片节点同时转圈生成 → 全部出图 | 「Step 3 并行生图（自动，等 2 分钟）」 |
| 25-35s | 每个图片节点拖出视频节点，全部生成 | 「Step 4 图转视频（自动，等 5 分钟）」 |
| 35-45s | 拖到时间轴，按顺序排好 | 「Step 5 拖进时间轴（手动，30 秒）」 |
| 45-52s | 点导出，进度条走完 | 「Step 6 导出 WebM」 |
| 52-58s | 成片对比 + 总耗时叠加 | 「全程 12 分钟，花费 18 元」 |
| 58-60s | Nomi logo + 下载提示 | 「开源 · 本地 · 双击即用 · 简介区」 |

**注**：录屏时记得放慢能看清节点的步骤，节奏跳着剪。

## 脚本 3 · 创作者人格款（45 秒，半身真人 + 屏幕画中画）

**目标**：建立信任，引导加微信 / Star。
**封面**：自己半身照 + 大字"我把它开源了"

**口播脚本（自然口语，可以背完了再录）**：

> [0-5s] 半身入镜
> "去年我开始做 AI 短视频。"
>
> [5-12s] 画面切到一堆软件 logo 切换录屏
> "前 3 个月每天工作 14 小时，但我发现一件事——大部分时间不是在创作，而是在 6 个 AI 工具之间复制粘贴。"
>
> [12-20s] 画面切到自己写代码的镜头 + Nomi 工作台界面
> "所以我自己做了一个工具，把写脚本、生图、生视频、剪辑放进了同一个工作台。"
>
> [20-28s] Nomi 画布生成动画 + 时间轴
> "AI 帮你拆镜头，并行生成，自动排进时间轴。本地跑，素材不上传。"
>
> [28-36s] 半身回到镜头
> "本来朋友让我商业化的。但是我自己就是创作者——我不想做一个收创作者月费的 SaaS。"
>
> [36-42s] 屏幕显示 GitHub 仓库 + Star 数
> "所以我把它开源了。叫 Nomi。今天发布桌面版，Mac、Windows 都能下。"
>
> [42-45s] 出微信号叠加
> "想进种子用户群的，主页加我。也欢迎来 Issues 砸我。"

- 视频末尾静态贴 5 秒：「GitHub 搜 aqm857886159/Nomi」
