01 / FLOW 创作流动

数据丝滑 流动，
而不是在工具之间复制粘贴。

剧本里的每一段提示词，都可以一键变成图片节点、视频节点。任何节点的片段，都能拖拽进入剪辑时间轴。

从一个想法到最终成片，你不需要切换七个软件。它们本来就该是一个东西。

所有素材，留在

04 / AGENT 助手 · 非主理

Agent 给建议，
你 才是导演。

Nomi 内置的 agent 可以陪你跑通一整段视频制作——它会提建议、补镜头、调色调，但最终的每一个选择都交给你。

AGENT · 04
/ 02 · 界面

三个面板，
一条河。

01
创作 / Script
从一段散文式的草稿出发，每一句都可以成为镜头。Nomi 会读懂上下文，给你提示词、节奏、灵感。
02
生成 / Canvas
一个无限画布。把图像和视频节点当作可编辑的卡片，连成你想要的故事。Agent 在右侧待命。
03
预览 / Timeline
所有节点都已经在时间轴上。拖、剪、叠、调速——传统剪辑该有的事情，都在这里发生。
/ 03 · 为什么这样设计

一条画布，
而不是
七个软件。

剧本、生成、剪辑，本来就该是一件事。把它们拆成三个软件，是工具的局限，不是创作的局限。Nomi 把它们压回同一张画布——剧本里的提示词直接长出图像和视频节点，节点的片段再直接落进剪辑轨。

"数据丝滑流动，而不是在工具之间复制粘贴。"

素材属于你，不属于云。整个工作流在本地跑：模型本地推理、文件本地存盘、不上传、不训练、不会某天因为"政策调整"消失。你的硬盘就是你的工作室。

Agent 是助理，不是替身。它会读上下文、补镜头、提建议，但 Nomi 不会替你做决定——所有的"选哪个"，留给你按下去。

/ 04 · 这里没有什么魔法

剧本 写出来 → 提示词 自动抽出 →
生成 图像 / 视频 节点 →
节点 拖入 时间轴 → 一条 成片。

一条画布，
从头到尾。
